# E-Commerce Database Schema

## Objective
Design and implement a relational database schema for an E-Commerce system.

## Tools Used
- MySQL Workbench
- MySQL

## Entities
- Users
- Categories
- Products
- Orders
- Order_Items
- Payments

## Key Concepts Applied
- DDL (CREATE, PRIMARY KEY, FOREIGN KEY)
- Normalization
- ER Diagram
- One-to-Many relationships

## Outcome
A well-structured and normalized database schema with proper relationships.
